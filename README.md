# `felixorosz.de`

**Note** - *frost.cat* used to be my domain name, thus the odd repository title

## Technologies

- Bun
- React
- Next.js
- TypeScript
- Tailwind CSS
- MDX (Remark, Rehype)
- Cloudflare Pages (for hosting, CDN & DNS)

## Get Started

Clone, then run the project using `bun`:

```
bun i
bun run dev
```
